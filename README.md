# כלי קודש למאק

> **Klei Kodesh for Mac** — a web-based (Office.js) Word add-in that brings Torah-editing
> tools to **Word for Mac**, where the Windows-only VSTO add-in cannot run.

תוסף ל-Word **למק**, מבוסס Office.js, לעורך התורני. נבנה בהשראת פרויקט
[כלי קודש](https://github.com/KleiKodesh/KleiKodeshProject) (שרץ על חלונות בלבד),
כפרויקט עצמאי המתאים למשתמשי מק. תודה לצוות כלי קודש על הרעיון והעידוד.

## מה התוסף כולל

- **חיפוש והחלפה (רגקס)** — חיפוש והחלפה לפי תבניות, מעבר ליכולות המובנות של וורד. עובד באופן עצמאי לחלוטין.
- **דיקטא וניקוד** — סימון טקסט במסמך וקבלת ניקוד אוטומטי משרת הניקוד של דיקטא, ישירות בחזרה למסמך. כולל ניקוי אוטומטי וסידור גרשיים.
- **אתרים תורניים** — פתיחת דיקטא, נקדן פרו, נקדנית, ספריא, HebrewBooks ואוצר החכמה בחלון בתוך וורד.

## התקנה (משתמש מק)

1. הורד את `manifest.xml` מהמאגר.
2. ב-Finder לחץ `Cmd+Shift+G` והדבק:
   `~/Library/Containers/com.microsoft.Word/Data/Documents/wef`
   (אם התיקייה `wef` לא קיימת — צור אותה.)
3. העתק את `manifest.xml` לתוך התיקייה. אל תדרוס מניפסטים אחרים שכבר שם.
4. סגור את Word לגמרי (`Cmd+Q`) ופתח מחדש.
5. בלשונית **בית (Home)** תופיע קבוצה **"כלי קודש"** עם כפתור — לחץ עליו.

## שימוש

**חיפוש והחלפה:** הקלד מה לחפש ובמה להחליף, ולחץ "החלף הכל". להסבר מתקדם — סמן "רגקס".

**ניקוד:** סמן טקסט עברי במסמך → לשונית "דיקטא וניקוד" → בחר סגנון → "נקד את הבחירה".

**אתרים:** לשונית "אתרים" — לחיצה פותחת את האתר בחלון בתוך וורד.

## אירוח

הקובץ `taskpane.html` מתארח דרך GitHub Pages של מאגר זה:
`https://lgolds311-code.github.io/klei-kodesh-mac/taskpane.html`.
המניפסט מצביע לשם. אם מארחים במקום אחר — יש לעדכן את כתובות ה-https שב-`manifest.xml`.

## הערות ומגבלות

- **הכפתור חובה:** המניפסט כולל `VersionOverrides` עם כפתור ברצועת הכלים. בלי זה התוסף לא מופיע ב-Word למק.
- **ניקוד אוטומטי:** מסתמך על שרת הניקוד הציבורי של דיקטא. שינוי/חסימה בצד דיקטא עלולים להשפיע; במקרה כזה התוסף פותח את נקדן פרו בחלון כגיבוי.
- **סינוני אינטרנט (נטפרי/אתרוג):** ייתכן שכתובת האירוח וחלק מהאתרים יידרשו אישור בסינון.
- **לא נכלל:** עיצוב תורני עמוק וקורא קיוויקס — דורשים יכולות VSTO שאינן קיימות ב-Office.js במק.
